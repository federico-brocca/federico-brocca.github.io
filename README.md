## Come pubblicare su GitHub Pages

1. Crea un repo chiamato `tuo-username.github.io`
2. Carica tutti i file in questo repo
3. Vai su **Settings → Pages → Source: main branch / root**
4. Il sito è live su `https://tuo-username.github.io`
